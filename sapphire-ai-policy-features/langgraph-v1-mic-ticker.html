LangGraph v1 & MIC Ticker Matching

A dependency and platform maintenance pass: LangChain/LangGraph upgraded to v1.0 with idiomatic patterns, exact MIC-coded ticker matching added across company search, and Docker support for marimo notebooks in local development.

Dependency Updates

  • LangChain/LangGraph v1.0: upgraded from v0.x, refactored to idiomatic v1 patterns (StateGraph, START/END nodes), with graph state migrated to TypedDict and Annotated.
  • Pre-commit hooks and production requirements regenerated from the updated lockfile.

Company Search & Ticker Matching

Exact matching added for MIC-coded tickers (e.g. AAPL:XNAS) across autocomplete, search, and the company financials tool, with case-insensitive handling. The scorecard repository also picked up a select_related optimization for its company foreign key.

Development Tools

Added Docker support for marimo, an interactive Python notebook environment, configured to work with the Django ORM and project dependencies — opt-in and with no effect on production deployments.

Reliability

All existing tests updated for LangGraph v1 compatibility, with new tests for MIC-coded ticker matching and scorecard repository query optimization.

Breaking (internal only): graph construction patterns changed significantly under the v1 migration, though external chatbot behavior is unaffected.

Data Handling Still Applies

A dependency upgrade and search-matching improvement — no change to what data any tool may access. See Data Security.

LangGraph v1 และการจับคู่ Ticker แบบ MIC

การบำรุงรักษา dependency และแพลตฟอร์ม: อัปเกรด LangChain/LangGraph เป็น v1.0 พร้อมรูปแบบการเขียนโค้ดที่เป็นมาตรฐาน (idiomatic) เพิ่มการจับคู่ ticker แบบรหัส MIC ที่แม่นยำในระบบค้นหาบริษัททั้งหมด และรองรับ Docker สำหรับ marimo notebook ในการพัฒนาบนเครื่อง local

การอัปเดต Dependency

  • LangChain/LangGraph v1.0: อัปเกรดจาก v0.x ปรับโครงสร้างใหม่ให้เป็นไปตามรูปแบบมาตรฐานของ v1 (StateGraph, node START/END) โดยย้าย graph state ไปใช้ TypedDict และ Annotated
  • Pre-commit hooks และ production requirements ถูกสร้างใหม่จาก lockfile ที่อัปเดตแล้ว

การค้นหาบริษัทและการจับคู่ Ticker

เพิ่มการจับคู่แบบแม่นยำสำหรับ ticker ที่มีรหัส MIC (เช่น AAPL:XNAS) ในระบบ autocomplete การค้นหา และเครื่องมือข้อมูลการเงินบริษัท โดยไม่สนตัวพิมพ์เล็ก/ใหญ่ นอกจากนี้ scorecard repository ยังได้รับการปรับปรุงประสิทธิภาพ select_related สำหรับ foreign key ของบริษัทด้วย

เครื่องมือสำหรับการพัฒนา

เพิ่มการรองรับ Docker สำหรับ marimo ซึ่งเป็น environment สำหรับ Python notebook แบบ interactive โดยตั้งค่าให้ทำงานร่วมกับ Django ORM และ dependency ของโปรเจกต์ได้ — เป็นแบบ opt-in และไม่มีผลกระทบต่อการ deploy บน production

ความน่าเชื่อถือ

อัปเดต test ที่มีอยู่เดิมทั้งหมดให้เข้ากันได้กับ LangGraph v1 พร้อมเพิ่ม test ใหม่สำหรับการจับคู่ ticker แบบรหัส MIC และการปรับปรุงประสิทธิภาพ query ของ scorecard repository

Breaking change (เฉพาะภายใน): รูปแบบการสร้าง graph เปลี่ยนแปลงอย่างมากจากการย้ายไปใช้ v1 แต่พฤติกรรมของแชทบอทที่ผู้ใช้ภายนอกเห็นไม่ได้รับผลกระทบ

หลักการจัดการข้อมูลยังคงบังคับใช้

การอัปเกรด dependency และการปรับปรุงการจับคู่การค้นหา — ไม่มีการเปลี่ยนแปลงต่อข้อมูลที่เครื่องมือใด ๆ สามารถเข้าถึงได้ ดูเพิ่มเติมที่ Data Security

LangGraph v1 နှင့် MIC Ticker ကိုက်ညီမှု

Dependency နှင့် platform ပြုပြင်ထိန်းသိမ်းမှု: LangChain/LangGraph ကို idiomatic pattern များနှင့်အတူ v1.0 သို့ အဆင့်မြှင့်ခဲ့ပြီး၊ ကုမ္ပဏီရှာဖွေမှု တစ်ခုလုံးတွင် exact MIC-coded ticker ကိုက်ညီမှု ထည့်သွင်းခဲ့ကာ local development တွင် marimo notebook များအတွက် Docker ပံ့ပိုးမှုကိုလည်း ထည့်သွင်းခဲ့သည်။

Dependency အပ်ဒိတ်များ

  • LangChain/LangGraph v1.0: v0.x မှ အဆင့်မြှင့်ခဲ့ပြီး v1 idiomatic pattern များ (StateGraph, START/END node) သို့ ပြန်လည်ဖွဲ့စည်းခဲ့သည်၊ graph state ကို TypedDict နှင့် Annotated သို့ ပြောင်းရွှေ့ခဲ့သည်။
  • Pre-commit hook များနှင့် production requirement များကို အပ်ဒိတ်လုပ်ထားသော lockfile မှ ပြန်လည်ထုတ်လုပ်ခဲ့သည်။

ကုမ္ပဏီရှာဖွေမှုနှင့် Ticker ကိုက်ညီမှု

MIC-coded ticker များ (ဥပမာ AAPL:XNAS) အတွက် exact matching ကို autocomplete၊ search နှင့် company financials tool တစ်လျှောက် ထည့်သွင်းခဲ့ပြီး စာလုံးအကြီးအသေး မခွဲခြားဘဲ ကိုင်တွယ်နိုင်သည်။ scorecard repository သည်လည်း ၎င်း၏ ကုမ္ပဏီ foreign key အတွက် select_related optimization ရရှိခဲ့သည်။

ဖွံ့ဖြိုးတိုးတက်ရေး ကိရိယာများ

Django ORM နှင့် project dependency များနှင့် အလုပ်လုပ်နိုင်ရန် ချိန်ညှိထားသော interactive Python notebook environment ဖြစ်သည့် marimo အတွက် Docker ပံ့ပိုးမှု ထည့်သွင်းခဲ့သည် — opt-in ဖြစ်ပြီး production deployment များအပေါ် မည်သည့်အကျိုးသက်ရောက်မှုမျှ မရှိပါ။

ယုံကြည်စိတ်ချရမှု

ရှိပြီးသား test အားလုံးကို LangGraph v1 နှင့် ကိုက်ညီစေရန် အပ်ဒိတ်လုပ်ခဲ့ပြီး MIC-coded ticker ကိုက်ညီမှုနှင့် scorecard repository query optimization အတွက် test အသစ်များ ထည့်သွင်းခဲ့သည်။

Breaking (အတွင်းပိုင်းသာ): v1 migration အောက်တွင် graph တည်ဆောက်ပုံများ သိသိသာသာ ပြောင်းလဲခဲ့သော်လည်း ပြင်ပ chatbot အပြုအမူသည် မည်သို့မျှ ထိခိုက်ခြင်း မရှိပါ။

ဒေတာ ကိုင်တွယ်မှု စည်းမျဉ်းများ ဆက်လက် သက်ရောက်

Dependency အဆင့်မြှင့်ခြင်းနှင့် search-matching တိုးတက်မှု — မည်သည့်ကိရိယာမျှ ရယူနိုင်သော ဒေတာအပေါ် ပြောင်းလဲမှု မရှိပါ။ Data Security ကို ကြည့်ပါ။
