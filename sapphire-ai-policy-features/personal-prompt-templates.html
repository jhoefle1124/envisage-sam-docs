Personal Prompt Templates

Alongside the admin-managed template library, users can now build and organize their own personal prompt templates — with categories, drag-and-drop ordering, and a one-click \"Save as Template\" straight from the chat editor.

What's New

  • User prompt templates and categories: personal templates with title, content, and category assignment, plus custom categories with icons.
  • Drag-and-drop ordering: sortable categories and templates within categories, managed through a dedicated interface.
  • Save as Template: a button in the chat editor saves the current prompt directly as a personal template.
  • Variable insertion: a toolbar button inserts template variables like {{company_name}} without hand-typing the braces.
  • Navigation update: the SAM AI section became a collapsible dropdown menu in both the main and admin sidenavs.

How It Relates to Prompt Studio

Personal templates are user-owned and separate from the admin-managed library covered on Administrative Management of Prompt Writing. Global templates can also be saved into a user's personal collection with a "Save to My Templates" button.

Reliability

1,454 lines of new tests covering views, forms, URLs, permissions, and edge cases. The models and views packages were split for maintainability while preserving backward compatibility through their existing import paths.

Data Handling Still Applies

Personal templates follow the same rule as the shared library: structure and placeholders are Tier 1 boilerplate, but don't hardcode a real client name or engagement detail into the stored template text — see Prompt Templates and Data Security.

เทมเพลตพรอมป์ต์ส่วนตัว

นอกเหนือจากคลังเทมเพลตที่ผู้ดูแลระบบจัดการ ผู้ใช้งานสามารถสร้างและจัดระเบียบเทมเพลตพรอมป์ต์ส่วนตัวของตนเองได้แล้ว — พร้อมหมวดหมู่ การจัดเรียงลำดับแบบลากวาง และปุ่ม "บันทึกเป็นเทมเพลต" แบบคลิกเดียวจากตัวแก้ไขข้อความในหน้าแชทได้โดยตรง

มีอะไรใหม่

  • เทมเพลตพรอมป์ต์และหมวดหมู่ของผู้ใช้: เทมเพลตส่วนตัวที่มีชื่อเรื่อง เนื้อหา และการกำหนดหมวดหมู่ พร้อมทั้งหมวดหมู่ที่กำหนดเองซึ่งมีไอคอนประกอบ
  • การจัดเรียงลำดับแบบลากวาง: จัดเรียงหมวดหมู่และเทมเพลตภายในแต่ละหมวดหมู่ได้ ผ่านหน้าจอที่ออกแบบมาโดยเฉพาะ
  • บันทึกเป็นเทมเพลต: ปุ่มในตัวแก้ไขข้อความของหน้าแชทที่บันทึกพรอมป์ต์ปัจจุบันเป็นเทมเพลตส่วนตัวได้โดยตรง
  • การแทรกตัวแปร: ปุ่มบนแถบเครื่องมือใช้แทรกตัวแปรของเทมเพลต เช่น {{company_name}} โดยไม่ต้องพิมพ์วงเล็บปีกกาเอง
  • การปรับปรุงเมนูนำทาง: ส่วน SAM AI เปลี่ยนเป็นเมนู dropdown แบบยุบ/ขยายได้ ทั้งใน sidenav หลักและ sidenav ของผู้ดูแลระบบ

ความเกี่ยวข้องกับ Prompt Studio

เทมเพลตส่วนตัวเป็นของผู้ใช้แต่ละคน และแยกออกจากคลังเทมเพลตที่ผู้ดูแลระบบจัดการ ซึ่งกล่าวถึงในหน้า การบริหารจัดการการเขียนพรอมป์ต์สำหรับผู้ดูแลระบบ นอกจากนี้ เทมเพลตส่วนกลางยังสามารถบันทึกลงในคลังส่วนตัวของผู้ใช้ได้ด้วยปุ่ม "บันทึกลงเทมเพลตของฉัน"

ความน่าเชื่อถือ

เทสต์ใหม่ 1,454 บรรทัด ครอบคลุม views, forms, URLs, permissions และกรณี edge case ต่าง ๆ แพ็กเกจ models และ views ถูกแยกออกจากกันเพื่อให้ดูแลรักษาง่ายขึ้น ขณะที่ยังคงความเข้ากันได้แบบย้อนหลัง (backward compatibility) ผ่าน import path เดิมที่มีอยู่

ข้อกำหนดการจัดการข้อมูลยังคงบังคับใช้

เทมเพลตส่วนตัวยังคงต้องปฏิบัติตามกฎเดียวกันกับคลังเทมเพลตส่วนกลาง กล่าวคือ โครงสร้างและ placeholder ต่าง ๆ ถือเป็น Tier 1 boilerplate แต่ห้าม hardcode ชื่อลูกค้าจริงหรือรายละเอียดของงานลงในข้อความเทมเพลตที่บันทึกไว้ — ดู Prompt Templates และ Data Security

SAM တွင် အသစ်ထွက်ရှိ · ဝန်ဆောင်မှု လမ်းညွှန်

ကိုယ်ပိုင် Prompt Template များ

admin managed template library နှင့်အတူ အသုံးပြုသူများသည် မိမိကိုယ်ပိုင် prompt template များကို ယခုအခါ တည်ဆောက်ပြီး စီစဉ်နိုင်ပြီ — category များ၊ drag-and-drop စီစဉ်မှု နှင့် chat editor မှ တိုက်ရိုက် တစ်ချက်နှိပ် "Save as Template" ပါဝင်သည်။

အသစ်များ

  • အသုံးပြုသူ prompt template များနှင့် category များ: ခေါင်းစဉ်၊ အကြောင်းအရာနှင့် category သတ်မှတ်ချက်ပါသော ကိုယ်ပိုင် template များ၊ icon ပါသော custom category များနှင့်အတူ
  • Drag-and-drop စီစဉ်မှု: category များနှင့် category အတွင်းရှိ template များကို စီစဉ်နိုင်ပြီး၊ သီးသန့် interface မှတစ်ဆင့် စီမံနိုင်သည်
  • Save as Template: chat editor ရှိ ခလုတ်တစ်ခုက လက်ရှိ prompt ကို ကိုယ်ပိုင် template အဖြစ် တိုက်ရိုက် သိမ်းဆည်းပေးသည်
  • Variable ထည့်သွင်းခြင်း: toolbar ခလုတ်တစ်ခုက {{company_name}} ကဲ့သို့ template variable များကို ကွင်းစကွင်းပိတ်များ ကိုယ်တိုင်ရိုက်စရာမလိုဘဲ ထည့်သွင်းပေးသည်
  • Navigation ပြင်ဆင်မှု — SAM AI အပိုင်းသည် main နှင့် admin sidenav နှစ်ခုစလုံးတွင် ခေါက်/ဖြန့်နိုင်သော dropdown menu အဖြစ် ပြောင်းလဲသွားသည်

Prompt Studio နှင့် ဆက်စပ်ပုံ

ကိုယ်ပိုင် template များသည် အသုံးပြုသူပိုင်ဆိုင်ပြီး Administrative Management of Prompt Writing တွင် ဖော်ပြထားသော admin managed library နှင့် သီးခြားစီ ရှိသည်။ Global template များကိုလည်း "Save to My Templates" ခလုတ်ဖြင့် အသုံးပြုသူ၏ ကိုယ်ပိုင် collection ထဲသို့ သိမ်းဆည်းနိုင်သည်။

ယုံကြည်စိတ်ချရမှု

test အသစ် line 1,454 လိုင်း — views, forms, URLs, permissions နှင့် edge case များကို လွှမ်းခြုံသည်။ models နှင့် views package များကို ထိန်းသိမ်းရလွယ်ကူစေရန် ခွဲထုတ်ထားပြီး၊ လက်ရှိ import path များမှတစ်ဆင့် backward compatibility ကို ဆက်လက် ထိန်းသိမ်းထားသည်။

ဒေတာကိုင်တွယ်မှု စည်းမျဉ်းများ ဆက်လက်သက်ရောက်

ကိုယ်ပိုင် template များသည် library ဘုံနှင့် စည်းမျဉ်းတူညီစွာ လိုက်နာရသည် — structure နှင့် placeholder များသည် Tier 1 boilerplate ဖြစ်သော်လည်း၊ သိမ်းဆည်းထားသော template စာသားထဲသို့ တကယ့်ဖောက်သည်နာမည် သို့မဟုတ် engagement အသေးစိတ်ကို hardcode မလုပ်ရ — Prompt Templates နှင့် Data Security ကို ကြည့်ပါ။
