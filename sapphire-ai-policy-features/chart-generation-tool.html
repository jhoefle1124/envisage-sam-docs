New in SAM · Feature Guide

Chart Generation Tool

SAM can now draw. A new chart generation tool lets the AI build interactive Highcharts visualizations directly in the chat — 14 chart types, automatic dark/light theming, and real-time rendering with no page refresh.

14 Chart Types

Line, bar, column, pie, donut, area, scatter, bubble, heatmap, treemap, polar, waterfall, funnel, organization, and sankey diagrams — covering everything from a simple price trend to a full org chart or flow diagram.

Architecture

ModulePurpose
tool.pyMain LangChain tool implementation
config_builder.pyHighcharts configuration builder per chart type
data_processors.pyData transformation and processing
validators.pyInput validation and error handling
themes.pyDark and light theme configurations

Charts render directly in the chat interface via a dedicated chart_renderer.js, with automatic theme switching and support for chart persistence in conversation history — a charts field was added to the chat message model to store configurations.

Reliability

92% test coverage across 828 test cases, covering every chart type and a wide range of edge cases.

Data Handling Still Applies

A chart is only as sensitive as the data plotted on it. A chart of public price trends is Tier 1; a chart built from a client portfolio's holdings is Tier 3, per Data Security — treat a saved or shared chart the same as the underlying numbers.

ฟีเจอร์ใหม่ใน SAM · คู่มือฟีเจอร์

เครื่องมือสร้างกราฟ

ตอนนี้ SAM วาดภาพได้แล้ว เครื่องมือสร้างกราฟใหม่นี้ช่วยให้ AI สร้างกราฟ Highcharts แบบ interactive ได้โดยตรงในหน้าแชท — รองรับกราฟ 14 ประเภท ปรับธีมมืด/สว่างอัตโนมัติ และแสดงผลแบบเรียลไทม์โดยไม่ต้องรีเฟรชหน้า

กราฟ 14 ประเภท

กราฟเส้น กราฟแท่งแนวนอน กราฟแท่งแนวตั้ง กราฟวงกลม กราฟโดนัท กราฟพื้นที่ กราฟกระจาย (scatter) กราฟฟอง (bubble) heatmap treemap กราฟเชิงขั้ว (polar) waterfall funnel แผนผังองค์กร (organization) และ sankey diagram — ครอบคลุมตั้งแต่แนวโน้มราคาแบบง่าย ๆ ไปจนถึงผังองค์กรเต็มรูปแบบหรือแผนผังกระบวนการ

สถาปัตยกรรม

โมดูลวัตถุประสงค์
tool.pyการทำงานหลักของ LangChain tool
config_builder.pyตัวสร้าง configuration ของ Highcharts สำหรับแต่ละประเภทกราฟ
data_processors.pyการแปลงและประมวลผลข้อมูล
validators.pyการตรวจสอบ input และจัดการข้อผิดพลาด
themes.pyการตั้งค่าธีมมืดและสว่าง

กราฟจะแสดงผลโดยตรงในหน้าต่างแชทผ่าน chart_renderer.js โดยเฉพาะ พร้อมสลับธีมอัตโนมัติ และรองรับการบันทึกกราฟไว้ในประวัติการสนทนา — มีการเพิ่มฟิลด์ charts ลงในโมเดลข้อความแชทเพื่อเก็บ configuration เหล่านี้

ความน่าเชื่อถือ

ครอบคลุมการทดสอบ 92% จากการทดสอบ 828 test case ครอบคลุมกราฟทุกประเภทและกรณี edge case ที่หลากหลาย

หลักการจัดการข้อมูลยังคงบังคับใช้

กราฟจะมีความอ่อนไหวเท่ากับข้อมูลที่นำมาพล็อตเท่านั้น กราฟแนวโน้มราคาสาธารณะจัดเป็น Tier 1 ส่วนกราฟที่สร้างจากการถือครองในพอร์ตของลูกค้าจัดเป็น Tier 3 ตามที่ระบุใน Data Security — ควรปฏิบัติต่อกราฟที่บันทึกหรือแชร์เหมือนกับตัวเลขต้นทางที่อยู่เบื้องหลัง

SAM တွင် အသစ်ထွက်ရှိ · ဖီချာလမ်းညွှန်

ဂရပ်ဖ် ဖန်တီးခြင်း ကိရိယာ

ယခုအခါ SAM သည် ပုံဆွဲနိုင်ပါပြီ။ ဂရပ်ဖ် ဖန်တီးခြင်း ကိရိယာအသစ်က AI ကို chat ထဲတွင် တိုက်ရိုက် interactive Highcharts ပုံများ ဖန်တီးနိုင်စေသည် — ဂရပ်ဖ် အမျိုးအစား ၁၄ မျိုး၊ dark/light theme အလိုအလျောက် ပြောင်းခြင်း နှင့် စာမျက်နှာ ပြန်မဆွဲရဘဲ real-time ပြသခြင်း။

ဂရပ်ဖ် အမျိုးအစား ၁၄ မျိုး

မျဉ်းဂရပ်ဖ်၊ bar ဂရပ်ဖ်၊ column ဂရပ်ဖ်၊ pie ဂရပ်ဖ်၊ donut ဂရပ်ဖ်၊ area ဂရပ်ဖ်၊ scatter ဂရပ်ဖ်၊ bubble ဂရပ်ဖ်၊ heatmap၊ treemap၊ polar ဂရပ်ဖ်၊ waterfall၊ funnel၊ organization chart နှင့် sankey diagram တို့ — ရိုးရှင်းသော ဈေးနှုန်း လမ်းကြောင်းမှသည် org chart အပြည့်အစုံ သို့မဟုတ် flow diagram အထိ လွှမ်းခြုံသည်။

ဖွဲ့စည်းပုံ

မော်ဂျူးရည်ရွယ်ချက်
tool.pyLangChain tool အဓိက အကောင်အထည်ဖော်မှု
config_builder.pyဂရပ်ဖ် အမျိုးအစားအလိုက် Highcharts configuration တည်ဆောက်သော အစိတ်အပိုင်း
data_processors.pyဒေတာ ပြောင်းလဲခြင်းနှင့် ပြုပြင်ခြင်း
validators.pyInput စစ်ဆေးခြင်းနှင့် error ကိုင်တွယ်ခြင်း
themes.pyDark နှင့် light theme ဆက်တင်များ

ဂရပ်ဖ်များကို chat interface ထဲတွင် chart_renderer.js အထူးမှတစ်ဆင့် တိုက်ရိုက်ပြသပြီး၊ theme အလိုအလျောက် ပြောင်းခြင်းနှင့် conversation history တွင် ဂရပ်ဖ်ကို ဆက်လက်သိမ်းဆည်းနိုင်ခြင်းကို ပံ့ပိုးသည် — configuration များကို သိမ်းဆည်းရန် chat message model တွင် charts field ကို ထည့်သွင်းခဲ့သည်။

ယုံကြည်စိတ်ချရမှု

test case ၈၂၈ ခုအနက် 92% test coverage ရရှိပြီး ဂရပ်ဖ် အမျိုးအစားအားလုံးနှင့် edge case အမျိုးမျိုးကို လွှမ်းခြုံသည်။

ဒေတာ ကိုင်တွယ်မှု စည်းမျဉ်းများ ဆက်လက် သက်ရောက်

ဂရပ်ဖ်တစ်ခု၏ အရေးကြီးမှုသည် ၎င်းတွင် ဖော်ပြထားသော ဒေတာအပေါ်သာ မူတည်သည်။ အများပြည်သူ ဈေးနှုန်း လမ်းကြောင်း ဂရပ်ဖ်သည် Tier 1 ဖြစ်ပြီး ဖောက်သည် portfolio ၏ holdings မှ ဖန်တီးထားသော ဂရပ်ဖ်သည် Data Security အရ Tier 3 ဖြစ်သည် — သိမ်းဆည်းထားသော သို့မဟုတ် share လုပ်ထားသော ဂရပ်ဖ်ကို နောက်ကွယ်ရှိ ဂဏန်းများနှင့် အတူတူပင် ဆက်ဆံပါ။
