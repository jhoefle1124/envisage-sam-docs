SAM Feature Reference

SAM Feature မှတ်တမ်း

SAM Chatbot အတွက် ထုတ်ပြန်ခဲ့သော feature အားလုံး၏ သီးခြားလမ်းညွှန်တစ်ခုဖြစ်ပြီး စာရင်းရှည်တစ်ခုအစား အမျိုးအစားခြောက်မျိုးအဖြစ် အုပ်စုဖွဲ့ထားသည်။ အောက်ပါအမျိုးအစားအလိုက် ရှာဖွေပါ သို့မဟုတ် အထက်ပါ Menu ကို အသုံးပြုပါ။

Chat အင်တာဖေ့စ်နှင့် Prompt ကိရိယာများ

SAM Chatbot ကို အသုံးပြုသူများ prompt ရေးသားပုံနှင့် ဆက်သွယ်ပုံကို ပုံဖော်ပေးသော အရာအားလုံး — တမ်းပလိတ်များ၊ Prompt Studio နှင့် Chat UI
