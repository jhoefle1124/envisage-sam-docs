Portfolio Analyst: Scorecard Rankings

The portfolio analyst tool now pulls scorecard data directly into portfolio and watchlist queries, so you can rank companies by internal, external, or ESG factors — and check up to 50 companies at once instead of 3. This page explains what changed and how to use it.

What's New

Portfolio analyst can now answer factor-based ranking questions directly, without a separate lookup step. The new scorecard_rankings query type integrates scorecard data with portfolio and watchlist queries in a single request:

  • Integrated scorecard data: portfolio/watchlist queries automatically pull in scorecard data — no separate lookup needed.
  • Factor-based rankings: rank companies by internal, external, or ESG scorecard factors.
  • Comprehensive filtering: filter by scorecard type, detail level, and latest-only.

What You Can Ask SAM

Example

"Which portfolio companies are rated highest on internal factors?"

Example

"Show watchlist companies with their scorecard ratings."

Example

"Get top companies by ESG scorecard scores."

Each of these now resolves in a single tool call — SAM fetches the portfolio or watchlist, pulls in scorecard ratings for every company, and ranks the result, rather than looking up each company one at a time.

Batch Processing: 3 Companies to 50

The underlying batch fetcher went from handling 3 companies per query to 50 — and from one tool call per company to a single tool call for the whole batch.

BeforeAfter
3 companies per queryUp to 50 companies per query
N tool calls for N companies1 tool call for N companies
Single-query lookups onlySingle query, multiple queries, or direct company IDs

For an analyst asking about a 30-company watchlist, this is the difference between one fast response and 30 sequential lookups.

Filtering Options

FilterWhat It Controls
Scorecard typeInternal, external, or ESG factors — or a combination
Detail levelHow much scorecard detail comes back with each company
Latest-onlyRestrict results to the most recent scorecard rating per company, rather than full history

Reliability

No breaking changes. Full backward compatibility is maintained — existing portfolio and watchlist queries behave exactly as before. The new ranking and batch capabilities are additive, backed by a 100+ test suite covering standard queries, batch operations, and edge cases.

Data Handling Still Applies

Portfolio and watchlist composition, and the scorecard ratings attached to them, are SAM client data — the same tier as any other fund holding or decision record on the Data Security page.

Tier 3

Stays inside SAM

Portfolio composition, scorecard ratings, and rankings are processed only inside SAM and approved Claude workflows under the engagement's terms — never pasted into free tools or verification models.

Check before quoting externally

Summaries and commentary

If you're drafting commentary that references a ranking or score outside SAM, treat client and company identifiers the same way you would any other Tier 3 content — anonymize or route through the approved workflow.
