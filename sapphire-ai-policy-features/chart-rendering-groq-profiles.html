New in SAM · Feature Guide

Chart Rendering, Groq & Parameter Profiles

Fixes a chart-rendering race condition and a dark-mode theming gap, adds Groq as a new model provider, and introduces parameter profiles that tune the model's behavior based on the kind of task at hand.

Bug Fixes

  • Chart rendering race condition: fixed timing issues when charts render inside a streaming message.
  • Dark mode theming: complete overhaul to use native Highcharts themes instead of ad hoc overrides, fixing export-button hover states and chart title/subtitle spacing along the way.

New Features

Groq Provider

Added as a new LLM provider with its own configuration.

Parameter Profiles

Dynamic parameter sets tuned per use case: default, research, creative, coding, and analysis.

Assumptions Protocol

An explicit assumptions-and-estimations protocol added to the system prompt for more transparent reasoning.

Chart Guidance

Improved chart type selection and generation guidance for the model.

Reliability

Chart generator tests updated for the new theme system, 285 lines of new tests for parameter profiles, and enhanced LLM factory tests.

Data Handling Still Applies

A new model provider is still bound by the same Data Security tool-routing rules — Groq, like any other provider, only receives what the tier of the underlying content allows.

ใหม่ใน SAM · คู่มือฟีเจอร์

การเรนเดอร์กราฟ, Groq และโปรไฟล์พารามิเตอร์

แก้ไข race condition ในการเรนเดอร์กราฟและช่องว่างของธีมโหมดมืด เพิ่ม Groq เป็นผู้ให้บริการโมเดลรายใหม่ และนำเสนอโปรไฟล์พารามิเตอร์ที่ปรับพฤติกรรมของโมเดลตามประเภทของงานที่ทำอยู่

การแก้ไขข้อบกพร่อง

  • Race condition ในการเรนเดอร์กราฟ: แก้ไขปัญหาด้านจังหวะเวลาเมื่อกราฟถูกเรนเดอร์ภายในข้อความแบบ streaming
  • ธีมโหมดมืด: ปรับปรุงทั้งหมดให้ใช้ธีม Highcharts แบบ native แทนการ override เฉพาะกิจ พร้อมแก้ไข hover state ของปุ่มส่งออกและระยะห่างของหัวข้อ/หัวข้อรองของกราฟไปในตัว

ฟีเจอร์ใหม่

Groq Provider

เพิ่มเป็นผู้ให้บริการ LLM รายใหม่พร้อมการตั้งค่าของตัวเอง

โปรไฟล์พารามิเตอร์

ชุดพารามิเตอร์แบบไดนามิกที่ปรับตามกรณีใช้งาน ได้แก่ default, research, creative, coding และ analysis

Assumptions Protocol

เพิ่มขั้นตอนการระบุสมมติฐานและการประมาณการอย่างชัดเจนใน system prompt เพื่อการให้เหตุผลที่โปร่งใสยิ่งขึ้น

คำแนะนำการสร้างกราฟ

ปรับปรุงคำแนะนำสำหรับโมเดลในการเลือกประเภทกราฟและการสร้างกราฟ

ความน่าเชื่อถือ

อัปเดตชุดทดสอบ chart generator สำหรับระบบธีมใหม่ เพิ่มการทดสอบ 285 บรรทัดสำหรับโปรไฟล์พารามิเตอร์ และปรับปรุงการทดสอบ LLM factory

หลักการจัดการข้อมูลยังคงใช้บังคับ

ผู้ให้บริการโมเดลรายใหม่ยังคงอยู่ภายใต้กฎการเลือกเครื่องมือของ ความปลอดภัยข้อมูล เช่นเดิม — Groq เช่นเดียวกับผู้ให้บริการรายอื่น จะได้รับเฉพาะข้อมูลตามที่ระดับชั้นของเนื้อหานั้นอนุญาตเท่านั้น

SAM ရှိ အသစ် · Feature Guide

ဂရပ်ဖ်ဖော်ပြခြင်း၊ Groq နှင့် Parameter Profile များ

ဂရပ်ဖ်ဖော်ပြခြင်း race condition တစ်ခုနှင့် dark-mode theme ကွာဟမှုတစ်ခုကို ပြင်ဆင်ပြီး၊ Groq ကို model provider အသစ်တစ်ခုအဖြစ် ထည့်သွင်းကာ၊ လုပ်ဆောင်နေသည့်အလုပ်အမျိုးအစားပေါ်မူတည်၍ model ၏ အပြုအမူကို ချိန်ညှိပေးသော parameter profile များကို မိတ်ဆက်ပေးသည်။

ချို့ယွင်းချက် ပြင်ဆင်မှုများ

  • ဂရပ်ဖ်ဖော်ပြခြင်း race condition: streaming message အတွင်း ဂရပ်ဖ်များ ဖော်ပြသည့်အခါ timing ပြဿနာများကို ပြင်ဆင်ထားသည်။
  • Dark mode theme: ad hoc override များအစား native Highcharts theme များကို အသုံးပြုရန် အားလုံးကို ပြန်လည်ပြင်ဆင်ထားပြီး၊ export ခလုတ် hover state နှင့် ဂရပ်ဖ်ခေါင်းစဉ်/ခေါင်းစဉ်ခွဲ spacing ကိုပါ တစ်ပါတည်း ပြင်ဆင်ထားသည်။

ဝန်ဆောင်မှုအသစ်များ

Groq Provider

ကိုယ်ပိုင် configuration ပါသော LLM provider အသစ်အဖြစ် ထည့်သွင်းထားသည်။

Parameter Profile များ

အသုံးပြုမှုအမျိုးအစားအလိုက် ချိန်ညှိထားသော dynamic parameter set များ — default, research, creative, coding နှင့် analysis။

Assumptions Protocol

ပိုမိုပွင့်လင်းသော ကျိုးကြောင်းဆက်နွှယ်မှုအတွက် system prompt တွင် ရှင်းလင်းသော assumptions-and-estimations protocol ကို ထည့်သွင်းထားသည်။

ဂရပ်ဖ် လမ်းညွှန်ချက်

model အတွက် ဂရပ်ဖ်အမျိုးအစား ရွေးချယ်ခြင်းနှင့် ဂရပ်ဖ်ဖန်တီးခြင်း လမ်းညွှန်ချက်ကို ပိုမိုကောင်းမွန်အောင် ပြုလုပ်ထားသည်။

ယုံကြည်စိတ်ချရမှု

Theme စနစ်အသစ်အတွက် chart generator test များကို update လုပ်ထားပြီး parameter profile များအတွက် test line ၂၈၅ လိုင်း အသစ်ထည့်သွင်းထားကာ LLM factory test များကိုလည်း မြှင့်တင်ထားသည်။

ဒေတာကိုင်တွယ်မှု စည်းမျဉ်း ဆက်လက်သက်ရောက်

Model provider အသစ်တစ်ခုသည်လည်း ဒေတာလုံခြုံရေး ၏ tool-routing စည်းမျဉ်းများနှင့် ချည်နှောင်ဆက်လက်ရှိနေသည် — Groq သည် အခြား provider များကဲ့သို့ပင် underlying content ၏ tier က ခွင့်ပြုသည့်အရာကိုသာ ရရှိမည်ဖြစ်သည်။
