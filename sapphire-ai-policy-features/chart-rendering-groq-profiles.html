New in SAM · Feature Guide

Chart Rendering, Groq & Parameter Profiles

Fixes a chart-rendering race condition and a dark-mode theming gap, adds Groq as a new model provider, and introduces parameter profiles that tune the model's behavior based on the kind of task at hand.

Bug Fixes

  • Chart rendering race condition: fixed timing issues when charts render inside a streaming message.
  • Dark mode theming: complete overhaul to use native Highcharts themes instead of ad hoc overrides, fixing export-button hover states and chart title/subtitle spacing along the way.

New Features

Groq Provider

Added as a new LLM provider with its own configuration.

Parameter Profiles

Dynamic parameter sets tuned per use case: default, research, creative, coding, and analysis.

Assumptions Protocol

An explicit assumptions-and-estimations protocol added to the system prompt for more transparent reasoning.

Chart Guidance

Improved chart type selection and generation guidance for the model.

Reliability

Chart generator tests updated for the new theme system, 285 lines of new tests for parameter profiles, and enhanced LLM factory tests.

Data Handling Still Applies

A new model provider is still bound by the same Data Security tool-routing rules — Groq, like any other provider, only receives what the tier of the underlying content allows.
