Modern Chat UI Redesign

A full visual overhaul of the chat interface: bubble-style messages, a mobile-responsive layout with sticky header and footer, toast notifications for transient errors, and a cleaner Prompt Studio on both desktop and mobile.

What's New

  • Bubble-style layout: a modern chat bubble design with improved visual hierarchy, increased max-width (1280px) for larger screens, and default site font sizing for better readability.
  • Mobile responsiveness: sticky header and footer, fixed viewport issues, scrollable table wrappers so wide tables no longer overflow a chat bubble.
  • Prompt Studio improvements: collapsible placeholders and a cleaner mobile templates tab, plus five specific desktop UX refinements.
  • Toast notifications: transient errors now surface as toasts instead of inline error text.

Fixes

  • EasyMDE editor initialization failures are now handled gracefully instead of breaking the input.
  • Fixed a double-escaping bug in chat title renaming.
  • Keyboard no longer covers the input field after inserting a prompt on mobile.

Data Handling Still Applies

A visual redesign — no change to what data the chat interface sends or stores. Standard Data Security rules continue to apply.

ใหม่ใน SAM · คู่มือฟีเจอร์

การออกแบบ UI แชทใหม่ที่ทันสมัย

การปรับโฉมภาพรวมของหน้าจอแชททั้งหมด: ข้อความสไตล์ bubble เลย์เอาต์ที่รองรับมือถือพร้อม header และ footer แบบ sticky การแจ้งเตือนแบบ toast สำหรับข้อผิดพลาดชั่วคราว และ Prompt Studio ที่สะอาดตาขึ้นทั้งบนเดสก์ท็อปและมือถือ

สิ่งใหม่

  • เลย์เอาต์แบบ bubble: ดีไซน์ chat bubble ที่ทันสมัยพร้อมลำดับชั้นภาพที่ดีขึ้น เพิ่ม max-width (1280px) สำหรับหน้าจอขนาดใหญ่ และใช้ขนาดฟอนต์เริ่มต้นของเว็บไซต์เพื่อให้อ่านง่ายขึ้น
  • รองรับการใช้งานบนมือถือ: header และ footer แบบ sticky แก้ปัญหา viewport ที่ค้าง และ wrapper ตารางที่เลื่อนได้ ทำให้ตารางกว้างไม่ล้นออกจาก chat bubble อีกต่อไป
  • การปรับปรุง Prompt Studio: placeholder ที่ยุบ/ขยายได้ และแท็บเทมเพลตบนมือถือที่สะอาดตาขึ้น พร้อมการปรับปรุง UX บนเดสก์ท็อปอีก 5 จุด
  • การแจ้งเตือนแบบ toast: ข้อผิดพลาดชั่วคราวจะแสดงเป็น toast แทนข้อความแจ้งข้อผิดพลาดแบบฝังในหน้า

การแก้ไขบั๊ก

  • ความล้มเหลวในการเริ่มต้นตัวแก้ไข EasyMDE ตอนนี้ถูกจัดการอย่างเหมาะสม แทนที่จะทำให้ช่องกรอกข้อมูลใช้งานไม่ได้
  • แก้ไขบั๊กการ escape ซ้ำซ้อนในการเปลี่ยนชื่อหัวข้อแชท
  • แป้นพิมพ์จะไม่บังช่องกรอกข้อความอีกต่อไป หลังจากแทรกพรอมต์บนมือถือ

การจัดการข้อมูลยังคงเหมือนเดิม

เป็นการปรับโฉมด้านภาพเท่านั้น — ไม่มีการเปลี่ยนแปลงข้อมูลที่หน้าจอแชทส่งหรือจัดเก็บ กฎมาตรฐานของ ความปลอดภัยข้อมูล ยังคงมีผลบังคับใช้เช่นเดิม

SAM တွင် အသစ် · Feature လမ်းညွှန်

ခေတ်မီ Chat UI ပြန်လည်ဒီဇိုင်းဆွဲခြင်း

Chat interface ကို အပြည့်အစုံ visual ပြန်လည်ပြင်ဆင်ခြင်း — bubble စတိုင် message များ၊ sticky header နှင့် footer ပါဝင်သော mobile-responsive layout၊ ယာယီ error များအတွက် toast notification များ နှင့် desktop နှင့် mobile နှစ်မျိုးလုံးတွင် သန့်ရှင်းလှပသော Prompt Studio။

အသစ်များ

  • Bubble စတိုင် Layout: visual hierarchy ပိုမိုကောင်းမွန်သော ခေတ်မီ chat bubble ဒီဇိုင်း၊ ဖန်သားပြင်ကြီးများအတွက် max-width (1280px) တိုးမြှင့်ခြင်း နှင့် ပိုမိုဖတ်ရှုလွယ်ကူစေရန် site ၏ default font size ကို အသုံးပြုခြင်း။
  • Mobile Responsiveness: sticky header နှင့် footer၊ viewport ပြဿနာများ ပြင်ဆင်ခြင်း၊ ကျယ်ပြန့်သော table များသည် chat bubble မှ ပြင်ပသို့ မလွှမ်းနိုင်တော့စေရန် scroll လုပ်နိုင်သော table wrapper များ။
  • Prompt Studio တိုးတက်မှုများ: ချုံ့/ဖြန့်နိုင်သော placeholder များနှင့် ပိုသန့်ရှင်းသော mobile templates tab၊ ထို့ပြင် သီးသန့် desktop UX ပြင်ဆင်မှု ၅ ခု။
  • Toast Notification များ: ယာယီ error များသည် inline error စာသားအစား ယခုအခါ toast အဖြစ် ပေါ်လာသည်။

ပြင်ဆင်ချက်များ

  • EasyMDE editor စတင်ရာတွင် ပျက်ကွက်မှုများကို ယခုအခါ input ကို ချို့ယွင်းစေမည့်အစား ချောမွေ့စွာ ကိုင်တွယ်ပေးသည်။
  • Chat ခေါင်းစဉ် အမည်ပြောင်းရာတွင် ဖြစ်ပေါ်သော double-escaping bug ကို ပြင်ဆင်ပြီးပြီ။
  • Mobile တွင် prompt ထည့်သွင်းပြီးနောက် ကီးဘုတ်သည် input field ကို ဖုံးအုပ်ခြင်း မရှိတော့ပါ။

ဒေတာကိုင်တွယ်မှု ဆက်လက်သက်ရောက်နေဆဲ

Visual ပြန်လည်ဒီဇိုင်းဆွဲမှုတစ်ခုသာဖြစ်ပြီး — chat interface မှ ပို့ဆောင်သော သို့မဟုတ် သိမ်းဆည်းသော ဒေတာတွင် ပြောင်းလဲမှု မရှိပါ။ စံ ဒေတာလုံခြုံရေး စည်းမျဉ်းများ ဆက်လက် သက်ရောက်နေဆဲဖြစ်သည်။
