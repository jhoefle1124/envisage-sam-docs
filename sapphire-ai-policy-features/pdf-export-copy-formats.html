New in SAM · Feature Guide

PDF Export & Copy Formats

Full chat sessions can now be exported as professionally styled PDFs, complete with stock charts rendered as embedded images, alongside a new copy dropdown for plain text, rich text, or HTML.

PDF Export

  • Full-session PDF export with dark/light-mode-aware styling and proper markdown/code-block rendering.
  • Highcharts stock charts are converted to PNG images (via Playwright) and embedded directly in the PDF.
  • Smart page-break handling avoids cutting a message awkwardly across pages.

Copy Format Options

A new dropdown lets you copy a response as plain text, rich text, or HTML — useful depending on whether you're pasting into an email, a Word document, or a web page.

Reliability

25 files changed, 2,582 insertions. A 557-line test suite covers PDF generation across various chat content, and the feature gracefully handles chats with no charts at all.

Data Handling Still Applies

An exported PDF carries the same tier as the conversation it's generated from. A PDF built from a Tier 3 conversation is itself Tier 3 — handle and store it exactly as you would the original chat, per Data Security.

ใหม่ใน SAM · คู่มือฟีเจอร์

การส่งออก PDF และรูปแบบการคัดลอก

ขณะนี้สามารถส่งออกเซสชันแชททั้งหมดเป็นไฟล์ PDF ที่จัดรูปแบบอย่างมืออาชีพได้แล้ว พร้อมกราฟหุ้นที่แสดงผลเป็นภาพฝังอยู่ในเอกสาร นอกจากนี้ยังมีเมนูคัดลอกแบบใหม่สำหรับข้อความธรรมดา ข้อความมีรูปแบบ (rich text) หรือ HTML

การส่งออก PDF

  • ส่งออก PDF ของทั้งเซสชัน พร้อมสไตล์ที่รองรับทั้งโหมดมืดและโหมดสว่าง และการแสดงผล markdown/code-block ที่ถูกต้อง
  • กราฟหุ้นจาก Highcharts จะถูกแปลงเป็นภาพ PNG (ผ่าน Playwright) และฝังลงในไฟล์ PDF โดยตรง
  • ระบบจัดการการขึ้นหน้าใหม่อย่างชาญฉลาด ป้องกันไม่ให้ข้อความถูกตัดขาดระหว่างหน้าอย่างไม่เหมาะสม

ตัวเลือกรูปแบบการคัดลอก

เมนูแบบใหม่ช่วยให้คุณคัดลอกคำตอบเป็นข้อความธรรมดา ข้อความมีรูปแบบ หรือ HTML ได้ — มีประโยชน์ขึ้นอยู่กับว่าคุณกำลังจะวางลงในอีเมล เอกสาร Word หรือหน้าเว็บ

ความน่าเชื่อถือ

แก้ไข 25 ไฟล์ เพิ่มโค้ด 2,582 บรรทัด ชุดทดสอบความยาว 557 บรรทัดครอบคลุมการสร้าง PDF สำหรับเนื้อหาแชทหลากหลายรูปแบบ และฟีเจอร์นี้จัดการแชทที่ไม่มีกราฟเลยได้อย่างราบรื่น

หลักการจัดการข้อมูลยังคงมีผล

PDF ที่ส่งออกจะมีระดับชั้นเดียวกับบทสนทนาที่สร้างมันขึ้นมา PDF ที่สร้างจากบทสนทนา Tier 3 ก็ถือเป็น Tier 3 เช่นกัน — จัดเก็บและดูแลรักษาเช่นเดียวกับแชทต้นฉบับ ตามที่ระบุใน ความปลอดภัยของข้อมูล

PDF Export နှင့် Copy ပုံစံများ

ယခုအခါ chat session အပြည့်အစုံကို stock chart များအား ပုံရိပ်အဖြစ် ထည့်သွင်းထားသော PDF အဖြစ် ပရော်ဖက်ရှင်နယ်ပုံစံဖြင့် export လုပ်နိုင်ပြီဖြစ်သည်၊ ထို့အပြင် plain text၊ rich text သို့မဟုတ် HTML အဖြစ် copy လုပ်ရန် dropdown အသစ်တစ်ခုလည်း ပါဝင်လာသည်။

PDF Export

  • Session အပြည့်အစုံ PDF export — dark/light mode ကို သိရှိသော styling နှင့် markdown/code-block များကို မှန်ကန်စွာ ဖော်ပြခြင်းပါဝင်သည်။
  • Highcharts stock chart များကို PNG ပုံများအဖြစ် (Playwright မှတစ်ဆင့်) ပြောင်းလဲပြီး PDF ထဲသို့ တိုက်ရိုက် ထည့်သွင်းသည်။
  • စမတ် page-break စီမံခန့်ခွဲမှုက စာမျက်နှာများကြား message များ မလိုလားအပ်ဘဲ ဖြတ်တောက်မသွားစေရန် ကာကွယ်ပေးသည်။

Copy ပုံစံ ရွေးချယ်စရာများ

dropdown အသစ်တစ်ခုက အဖြေတစ်ခုကို plain text၊ rich text သို့မဟုတ် HTML အဖြစ် copy လုပ်ခွင့်ပြုသည် — email၊ Word document သို့မဟုတ် web page တစ်ခုထဲသို့ paste လုပ်မည်ဆိုသည်ပေါ်မူတည်၍ အသုံးဝင်သည်။

ယုံကြည်စိတ်ချရမှု

ဖိုင် 25 ဖိုင် ပြောင်းလဲထားပြီး 2,582 လိုင်း ထပ်ပေါင်းထားသည်။ line 557 လိုင်းပါ test suite သည် chat content အမျိုးမျိုးအတွက် PDF ဖန်တီးမှုကို ကာမိပြီး၊ ဤဝန်ဆောင်မှုသည် chart လုံးဝမပါသော chat များကိုလည်း ချောမွေ့စွာ ကိုင်တွယ်နိုင်သည်။

ဒေတာကိုင်တွယ်မှု စည်းမျဉ်းများ ဆက်လက်သက်ရောက်

Export လုပ်ထားသော PDF သည် ၎င်းကို ဖန်တီးထားသည့် စကားဝိုင်းနှင့် တူညီသော tier ကို ဆောင်ရသည်။ Tier 3 စကားဝိုင်းမှ ဖန်တီးထားသော PDF သည် ကိုယ်တိုင် Tier 3 ဖြစ်သည် — မူရင်း chat ကို ကိုင်တွယ်သကဲ့သို့ တိကျစွာ ကိုင်တွယ်ပြီး သိမ်းဆည်းပါ၊ ဒေတာလုံခြုံရေး အရ။
